# Ketoy Brand Identity

## 🎨 Brand Overview

**Ketoy** is a modern Server-Driven UI platform that enables seamless app management and dynamic screen configuration. The brand represents innovation, efficiency, and developer-friendly tools.

## 🔷 Logo

### Logo Design
The Ketoy logo features a circular sun/spinner design with radiating segments, symbolizing:
- **Growth & Energy**: The radiating lines represent expansion and dynamism
- **Connectivity**: The circular form suggests unity and seamless integration
- **Innovation**: The modern, geometric design reflects cutting-edge technology

### Logo Specifications
- **Shape**: Circle with 16 radiating segments
- **Center**: Solid circle (radius: 20% of total)
- **Segments**: Lines radiating from inner circle to outer edge
- **Style**: Minimalist, geometric, modern

### Logo Variations

#### Main Logo (Dark on Light)
- Background: White (#FFFFFF)
- Foreground: Black (#000000)
- Use: Headers, light backgrounds

#### Inverted Logo (Light on Dark)
- Background: Dark backgrounds
- Foreground: White (#FFFFFF)
- Use: Dark theme interfaces

## 🎭 Design Principles

### 1. Dark-First Design
- Primary interface uses dark theme
- Reduces eye strain for developers
- Modern, professional appearance

### 2. Clear Hierarchy
- Strong contrast between elements
- Consistent spacing system (4px, 8px, 16px, 24px, 32px)
- Clear visual grouping

### 3. Minimal & Functional
- Clean, uncluttered interface
- Focus on functionality
- Purposeful use of color

### 4. Developer-Friendly
- Familiar patterns (inspired by VS Code, GitHub)
- Clear error messages
- Helpful tooltips and placeholders

## 📐 Layout Guidelines

### Spacing Scale
- **xs**: 4px (0.25rem)
- **sm**: 8px (0.5rem)
- **md**: 16px (1rem)
- **lg**: 24px (1.5rem)
- **xl**: 32px (2rem)
- **2xl**: 48px (3rem)

### Border Radius
- **sm**: 4px (0.25rem) - Small elements
- **md**: 8px (0.5rem) - Buttons, inputs
- **lg**: 12px (0.75rem) - Cards
- **xl**: 16px (1rem) - Large cards
- **full**: 9999px - Circular elements

### Container Widths
- **Max width**: 1280px (80rem)
- **Content padding**: 16px mobile, 24px tablet, 32px desktop

## 🖼️ Icon Style

### Icon Characteristics
- **Style**: Outline (2px stroke)
- **Size**: 20px (default), 24px (large)
- **Color**: Inherits from text color
- **Source**: Heroicons or custom SVG
